# 国标麻将番型列表

基于 `server/src/fan/FanCalculator.ts` 实现，最低和牌门槛 **8 番**。

> **说明**
> - **番值** 为单次命中的计分值；部分番型可计多次（如"幺九刻"最多 4 次、"暗杠"每个 2 番）。
> - **排除** 为命中本番型后自动跳过不计分的番型（避免重复计分）。

## 积分计算规则

**支付额 = 番数（含花牌）+ 8**

| 和牌方式 | 付款方 | 金额 |
|---------|--------|------|
| 自摸（摸和） | 其余三家各付 | 支付额 |
| 点和 | 放炮者付 | 支付额 |
| 点和 | 另外两家各付 | 8 |

和牌者获得所有付款之和。无番和时支付额为 0 + 8 = 8。
